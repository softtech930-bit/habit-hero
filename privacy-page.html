Privacy Policy

Last Updated:

Habit Hero (“the App”) is an open-source Android application created for learning and personal development. This Privacy Policy explains how your data is handled.

1. Data Collection

The App does not collect, store, or transmit any personal data to the developer or any external servers.

The App does not use:

  • Analytics tools
  • Advertising services
  • Tracking technologies
  • Cloud services
  • Third-party SDKs that collect personal information

2. Local Storage Only

All data generated in the App — including habits, progress, notes, and settings — is stored only on your device using a local database. No data leaves your device.

3. Permissions

The App requests only minimal permissions necessary for its core functionality. It does not request unnecessary permissions.

4. Open Source

Habit Hero is an open-source project. Anyone may review the code to verify how data is processed and stored.

5. Changes to This Policy

This Privacy Policy may be updated when necessary. Any changes will be posted on this page.

6. Contact

If you have any questions, you may contact the developer through the GitHub repository.
